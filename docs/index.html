Why NebulaDB?

Performance First

Blazing-fast in-memory execution with advanced B-tree indexing and query optimization.

Reactive Queries

Live query support enables seamless reactivity in real-time apps and dashboards.

Type-Safe

TypeScript-native API delivers strong typing and developer confidence out-of-the-box.

Modular Core

Drop-in adapters and plugin architecture lets you build exactly what you need—no bloat.

Truly Cross-Platform

Runs everywhere—from Node.js to browsers and edge environments like Cloudflare Workers.

One Package

No dependencies, no nonsense. Just install and go—serverless simplicity.

Getting Started

Installation
npm install @nebula-db/nebula-db

Install once. Build anywhere. It's that simple.

Basic Example
import { createDatabase } from '@nebula-db/nebula-db';

const db = createDatabase();
const users = db.collection('users');

await users.insert({ name: 'Alice', age: 30 });
const result = await users.find({ age: { $gt: 20 } });
console.log(result);
