Getting Started

Installation

npm install @nebula-db/nebula-db

That's it! One package with everything you need to get started.

Basic Usage

import { createDatabase } from '@nebula-db/nebula-db';

// Create a database with in-memory storage
const db = createDatabase();

// Create a collection
const users = db.collection('users');

// Insert a document
await users.insert({ name: 'Alice', age: 30 });

// Query documents
const result = await users.find({ age: { $gt: 20 } });
console.log(result);
